Features

Everything you need, nothing you don't

🎯

Hold-Modifier Buttons

R and ZR act as modifier keys (Right ⌘, Right ⌥) that you hold down while pressing other buttons. Just like a real keyboard.

⚡

Combo Keys

Any button can fire a full key combination — B → ⌘+Z for undo, SL → ⌘+C for copy — without holding a modifier first.

🔄

Tap-to-Cycle App Switching

Each HOME tap advances the macOS app switcher (⌘+Tab) exactly once. Stop tapping for 0.8 seconds to select the app.

🎤

Hold-to-Talk Voice Input

SR holds Right ⌥ to trigger voice input via native HID event — press to start, release to stop. Works with SaySo and macOS dictation.

🕹️

Analog Stick Mapping

Push the stick left/right/up/down to trigger arrow keys in all four directions. Perfect for navigating AI code suggestions without touching the keyboard.

📡

Auto-Reconnect + Watchdog

If Bluetooth drops, the script waits and reconnects. Optional watchdog daemon reconnects at the Bluetooth level via blueutil.

🌐

Web Config UI

Cyberpunk-themed browser interface for visual configuration. Interactive SVG diagram, modal editing, real-time connection status.

📋

JSON Configuration

All mappings live in config.json — human-editable, version-controllable. No proprietary formats, no cloud dependency.
