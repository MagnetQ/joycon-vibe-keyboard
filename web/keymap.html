典型 Vibe Coding 工作流

用 Joy-Con 就能完成完整的 AI 编程循环，不用碰键盘：

R 按住说话（Typeless 语音输入 prompt） → A 发送 → 摇杆选选项 → A 确认 → + 接受补全 → B 撤销（如果改错了）

注意事项

• 摇杆 Y 轴硬件无输出（副厂 Joy-Con），因此用 X 轴左右替代上下
• 蓝牙断开后脚本会自动等待重连，无需手动重启
• 脚本路径：~/Documents/workspace-ai/code/joycon-vibe-keyboard/joycon_mapper.py
• 修改映射：运行 python3 config_server.py，浏览器打开 http://localhost:8766 可视化编辑
